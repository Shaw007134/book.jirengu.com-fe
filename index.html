本书介绍

饥人谷课件

本课件作者： 若愚、方方、Tom、禄永

自我介绍

  1. 个人信息：姓名、性别、所在地、在工作还是在上学、兴趣爱好等
  2. 学习目标、学习计划等

课程安排

1. 上课时间

统一协商

2. 上课形式

  • 上课记笔记
  • 课后完成作业、搜索阅读文章、写博客

3. 学习习惯

  • Google搜索题目或者知识点关键字
  • 查看 MDN 文档，技巧：搜索「关键词 MDN」
  • 查阅博客
    • 阮一峰 javascript.ruanyifeng.com
    • MDN
    • 饥人谷博客 blog.jirengu.com
  • 谁都不要相信，只相信自己动手的结果

4. 关于考勤

  • 有特殊原因，提前在群里 at 班主任
  • 迟到的同学表演节目
  • 无故缺勤或者迟到且不遵守惩罚规则连续5次，流放

5. 如何学习复习

  • 第二天可在官网看录屏
  • 任何问题问老师、找教导员
  • 上课时将重要知识点写到本子上
  • 为了让自己弄清楚，请写博客记录

6. 如何写作业

  • github pages
  • js.jirengu.com

7. 如何写博客

  1. github issue
  2. 简书
  3. 知乎

课堂之外

1. 如何提问

  1. 你会问问题吗？
  2. X-Y PROBLEM
  3. 展示问题的所有细节、尽量让帮你的人省事
  4. js.jirengu.com

2. 如何快速得到解答

  1. 在群里@所有的老师，10分钟内会得到解答
  2. 私聊没问题，回复可能没那么及时
  3. 周末或者晚上10:30之后，回复可能不及时
  4. 帮助别人收获的更多

找工作相关
