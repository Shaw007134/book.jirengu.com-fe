文件路径

相对路径

同一站点尽量使用相对路径。
假设文件结构如下：

- index.html
- images/
    logo.png
    background.png
- styles/
    app.css

<!-- index.html -->
<img src="images/logo.png">

/* app.css */
.container {
  background: url('../images/background.png');
}
